# flow-first-builder

A flow-first AI development skill for clarifying complex features and modules with Mermaid diagrams and zoomable HTML previews. Still actively evolving.

## What it does

- clarifies whether to start from a main flowchart or sub flowchart
- asks one key question per round
- forces exactly one diagram per response
- keeps Mermaid as the source of truth
- can generate zoomable HTML previews for dense diagrams
- blocks implementation until the logic is confirmed

## Files

- `SKILL.md`: main skill definition
- `SKILL_SPEC.md`: supporting specification notes
- `INTERACTION_EXAMPLE.md`: usage example
- `references/`: deeper reference material
- `scripts/`: helper scripts, including Mermaid HTML preview generation
